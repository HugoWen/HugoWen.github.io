Composer 背景

Composer 是 PHP 项目的依赖管理器，用来解决PHP项目没有快速引入三方库代码的情况。比较类似 Node 的 NPM。

Composer 组成

composer.json：项目安装文件
初始化文件，在文件内定义需要引入的依赖关系，包括包名和版本号。

composer.lock：锁文件
锁定项目中引入的三方依赖的报名和版本号。

SCP协议原理及使用
2018-05-21
Linux

SCP协议

SCP 协议，全称 Secure Copy 协议，中文名是 安全复制协议。是指在本地主机与远程主机或两台远程主机之间基于 Secure Shell (SSH) 协议传输文件的协议。

SCP 使用 SSH 完成身份认证以及数据传输。SCP 包含 SCP CLIENT（客户端） 和 SCP SERVER（服务端），客户端可以向服务端发送文件，也可以从服务端接收文件。

SCP 默认通过 TCP 端口 22 运行。
