MacOS配置LNMP开发环境
2018-05-18
MacOS

1. 安装 Homebrew

2. 安装 php7.2

brew install php

brew install php 默认为最新版本的php，当前是7.2版本。
现在已经不需要另外引入php的三方brew库，已经合并到官方库里了

安装成功后开启 php 服务 brew services start php

假如没有卸载系统预装的php环境，也没有关系，只需要调整一下路径，编辑 /etc/paths 文件，在最前面加入 /usr/local/Cellar
