####1. 安装 Homebrew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

####2. 安装 php7.2

brew install php

brew install php 默认为最新版本的php，当前是7.2版本
现在已经不需要另外引入php的三方brew库，已经合并到官方库里了

安装成功后开启 php 服务 brew services start php

假如没有卸载系统预装的php环境，也没有关系，只需要调整一下路径，编辑 /etc/paths 文件，在最前面加入 /usr/local/Cellar

####3. 安装 nginx

brew install nginx

安装成功后开启 nginx 服务 brew services start nginx

nginx 配置文件路径在 /usr/local/etc/nginx/servers 下

####4. 安装 mysql

brew install mysql

安装成功后开启 mysql 服务 mysql.server start

默认用户名为 root ，密码为空

####5. 安装 redis

brew install redis

安装成功后开启 redis 服务 brew services start redis

其他一些 php 扩展包可能需要使用 pecl 来安装，以安装 grpc 举例。

####1. 安装 pecl

####2. 安装 grpc 扩展

pecl install grpc

安装完成后会自动打开 grpc 扩展，不需要再手动修改 php.ini 文件。
